# OnlineTV

Автоматический IPTV плейлист с ежечасным обновлением.

## Ссылка для плеера

https://raw.githubusercontent.com/SlayerHasher/OnlineTV/main/playlist.m3u

## Особенности

- Автоматическое обновление каждый час
- Проверка работоспособности стримов
- Сохранение оригинальных категорий из источников
- Удаление дубликатов

## Источники

- Dimonovich FREE TV
- LoganetTV Playlists
- IPTV MEGA PLAYLIST

## Ручной запуск

Actions → Update Playlist → Run workflow
